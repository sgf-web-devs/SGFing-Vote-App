DeviceCamera



Description

This class provides an abstraction for device cameras.



Public Variables

ExposureBias Get or set the camera's exposure bias.
ExposureMode Get or set the camera's exposure mode.
FlashMode Get or set the camera's flash mode.
FocusMode Get or set the camera's focus mode.
FocusPoint Focus the camera to a viewport point.
Framerate Get or set the current framerate of the camera.
HorizontalFOV Horizontal view angle of the camera in degrees.
IsFlashSupported Does the camera support flash?
IsFrontFacing Is the camera front-facing?
IsTorchSupported Does the camera support torch?
MaxExposureBias Maximum exposure bias supported by the camera.
MaxZoomRatio Get the camera's maximum zoom ratio.
MinExposureBias Minimum exposure bias supported by the camera.
PhotoResolution Get or set the photo resolution of the camera.
PreviewResolution Get or set the preview resolution of the camera.
TorchEnabled Get or set the camera's torch mode.
VerticalFOV Vertical view angle of the camera in degrees.
ZoomRatio Get or set the camera's zoom ratio.


Static Variables

Cameras An array of all cameras for this device.
FrontCamera The default front camera for this device.
RearCamera The default rear camera for this device.
